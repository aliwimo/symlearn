Firefly Programming (``FFP``)
-----------------------------

Firefly programming (``FFP``) is an automatic programming method that uses the principles of symbolic regression to solve problems, for further information check this reference: `Ext. Link <https://ieeexplore.ieee.org/abstract/document/9302201/>`_.
